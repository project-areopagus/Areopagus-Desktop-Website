# Areopagus Website

The marketing website for [Areopagus](https://github.com/project-areopagus) — a privacy-first communication and creator platform.

## Hosting on GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Under "Source", select **Deploy from a branch**
4. Choose `main` branch, `/ (root)` folder
5. Click **Save**

Your site will be live at `https://<your-username>.github.io/<repo-name>/` within a few minutes.

## Local Development

Open `index.html` in your browser, or use VS Code with the [Live Server extension](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) for auto-refresh.

## File Structure

| File | Purpose |
|------|---------|
| `index.html` | All content, structure, and JavaScript (~940 lines) |
| `style.css` | All visual styling and component styles (~1800 lines) |
| `base.css` | CSS reset and design token definitions |

## Themes

The site includes 14 built-in themes across 4 groups (Areopagus, Mars Hill, Corporate, Homecoming). Themes are selectable via the palette icon in the header.
